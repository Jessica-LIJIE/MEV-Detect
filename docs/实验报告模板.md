## 1. 引言

### 1.1 研究背景

MEV、DEX 套利、L1-L2 价差与原子套利窗口。

### 1.2 相关工作

- MEVisor (NDSS 2025)：GA + GPU 单链 MEV  
- 本项目：PSO + 真多 GPU + L1-L2 + 可视化  

### 1.3 研究目标

1. 以可并行 PSO 替换 GA  
2. **真多 GPU** 验证粒子分片加速  
3. L1 Ethereum ↔ L2 Arbitrum 套利检测  
4. 构建可演示的可视化检测系统

## 2. 系统设计

### 2.1 架构图

【插入：监听 → 快照 → PSO（多 GPU）→ 策略 → 可视化】

### 2.2 向量化 PSO

- 搜索空间：$X = [Amount_{in}, Route_{L1}, Route_{L2}, Bridge_{path}]$  
- 矩阵 shape：`[num_particles, dim]`  
- 与 GA 差异：无交叉变异串行依赖  

### 2.3 多 GPU 方案

- 粒子按 rank 分片  
- 每轮 `all_reduce` + `broadcast` 同步 gbest  
- 【插图：多卡数据流】
