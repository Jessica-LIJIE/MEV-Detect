### **四项实验各自用什么数据**


| **实验**           | **数据**                      | **跑在哪**   |
| ---------------- | --------------------------- | --------- |
| **E1** PSO vs GA | Mock（多粒子数对比）                | 本地即可      |
| **E2** 多 GPU 加速  | **Mock 单条快照**               | **必须云多卡** |
| **E3** 跨层套利检测    | Mock 回放 + **Live 真实监听**     | 本地（RPC）   |
| **E4** 可视化       | Mock 结果 + Live 日志 + E1/E2 图 | 本地        |


所以：**E2 和 E3 是分开的**——多 GPU 证明并行能力用 Mock；跨层检测能力用 E3 的 Mock/live。
