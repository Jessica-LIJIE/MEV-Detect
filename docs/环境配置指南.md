# 环境配置指南

---

## 1. 双环境策略

本项目使用 **两套运行环境**，职责不同：

| 环境 | 用途 | 硬件 |
|------|------|------|
| **本地开发机** | 编码、Mock、监听、可视化、E1 实验 | Windows，CPU 或单卡 |
| **云 GPU 主机** | **E2 真多 GPU benchmark**（必须） | Linux，2～4× NVIDIA GPU |

老师要求多 GPU 实测，**E2 只能在云主机完成**，不能用本地 CPU 双进程代替。

---

## 2. 本地环境

### 2.1 要求

| 组件 | 最低 | 推荐 |
|------|------|------|
| Python | 3.10+ | 3.12 |
| 内存 | 8 GB | 16 GB |
| 磁盘 | 2 GB | 5 GB |
| GPU | 可选 | 单卡（E1 加速；E2 不依赖本地 GPU） |

### 2.2 创建虚拟环境

**Windows (CMD)：**

```cmd
cd MEV-Detect
python -m venv .venv
.venv\Scripts\activate.bat
pip install -r requirements.txt
pip install torch --index-url https://download.pytorch.org/whl/cpu
```

**macOS / Linux：**

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

有 NVIDIA GPU 且 PyTorch 支持时，到 https://pytorch.org 选择对应 CUDA 轮子。

### 2.3 验证

```bash
python scripts/check_env.py
python -c "import torch; print(torch.__version__, torch.cuda.is_available())"
```

> **说明：** RTX 50 系（sm_120）可能暂不被当前 PyTorch 支持，本地设 `DEVICE=cpu` 即可，不影响 E2 云主机实验。

## 3. 环境变量 `.env`

在项目根目录创建 `.env`（**勿提交 Git**），参考 `.env.example`：

```env
# Ethereum Mainnet
ETH_HTTP_URL=https://mainnet.infura.io/v3/YOUR_PROJECT_ID
ETH_WS_URL=wss://mainnet.infura.io/v3/YOUR_PROJECT_ID

# Arbitrum One
ARB_HTTP_URL=https://arbitrum-mainnet.infura.io/v3/YOUR_PROJECT_ID
ARB_WS_URL=wss://arbitrum-mainnet.infura.io/v3/YOUR_PROJECT_ID

# Infura 开启 API Key Secret 时必填
INFURA_API_KEY_SECRET=your_secret

DEVICE=cpu
```

### RPC 验证

```bash
python scripts/test_rpc.py
python -m pytest tests/test_listener.py::test_rpc_fetch_latest_snapshot -v
```

国内网络若 ETH WebSocket 超时，live 模式使用：

```bash
python main.py --live --http --duration 60
```

## 8. 云 GPU 多卡环境（E2 必做）

### 8.1 平台推荐

| 平台 | 参考价格 | 说明 |
|------|----------|------|
| [AutoDL](https://www.autodl.com) | RTX 4090 ≈ 2～2.5 元/小时/卡 | 学生优惠，预装 PyTorch 镜像 |
| 恒源云 | 类似 | 备选 |

选实例时选 **同一台机器多卡**（如 2×4090、4×4090），不要开两台单机。

### 8.2 租用步骤

1. 注册、充值（**10～20 元** 通常够用）  
2. 选镜像：**PyTorch 2.x + CUDA 12.x + Ubuntu**  
3. 选 GPU 数量：2 或 4  
4. SSH / Jupyter 进入，克隆项目：

```bash
git clone <your-repo-url> MEV-Detect
cd MEV-Detect
pip install -r requirements.txt
```

5. 验证多卡：

```bash
python -c "import torch; print(torch.cuda.device_count())"
nvidia-smi
```

6. 跑 E2（代码实现后）：

```bash
torchrun --nproc_per_node=2 scripts/run_ddp.py --mock --record-id snap_003
python scripts/benchmark_multi_gpu.py
```

7. **下载图表** `data/figures/` 到本地，**关机停费**。

### 8.4 常见问题

| 问题 | 处理 |
|------|------|
| `NCCL error` | 确认多卡同节点；设置 `export NCCL_IB_DISABLE=1` 有时可解 |
| 只用到 1 张卡 | 检查是否用 `torchrun` 启动，而非 `python` 单进程 |
| 连不上 GitHub | 本地上传 zip 或用 Gitee 镜像 |
