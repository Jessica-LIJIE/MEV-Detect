## 2. 阶段 1：真多 GPU

### 2.1 修改 `src/optimizer.py`

1. 在 `PSOOptimizer.search()` 每轮 fitness 计算后增加 `_sync_gbest()`  
2. 使用 `torch.distributed.all_reduce` / `broadcast` 同步全局最优  
3. `create_pso_optimizer()` 按 `world_size` 划分每 rank 粒子数  

**硬性约束不变：** 粒子级更新仍不得使用 Python `for`。

### 2.2 新增 `scripts/run_ddp.py`

```python
# 伪代码结构
def main():
    rank, local_rank, world_size = init_distributed()
    device = f"cuda:{local_rank}"
    snapshot = load_snapshot(args)  # rank 0 加载，broadcast 元数据
    optimizer = create_pso_optimizer(..., rank=rank, world_size=world_size)
    result = optimizer.search(fitness_fn, ...)
    if rank == 0:
        save_result(result)
    dist.destroy_process_group()
```

### 2.3 新增 `scripts/benchmark_multi_gpu.py`

- 循环 `nproc_per_node` ∈ {1, 2, 4}  
- 固定 `seed`、固定快照  
- 输出 JSON + 调用 matplotlib 出图  

### 2.4 验收

```bash
# 云主机上
torchrun --nproc_per_node=2 scripts/run_ddp.py --mock --record-id snap_003
python scripts/benchmark_multi_gpu.py
```

## 3. 阶段 2：可视化与扩展

### 3.1 Streamlit 面板 `viz/dashboard.py`

依赖：`streamlit`（加入 `requirements.txt`）

页面结构建议：

1. **侧边栏**：数据源选择（Mock / Live 日志 / Benchmark JSON）  
2. **主区 Tab 1**：价差时序图（matplotlib / plotly）  
3. **Tab 2**：触发记录表（pandas DataFrame）  
4. **Tab 3**：策略详情与成本分解  
5. **Tab 4**：E1/E2 性能图（读 `data/figures/`）  

### 3.2 结果持久化

在 `main.py` 回调中追加写入：

```text
data/results/live_log.jsonl   # 每行一条 JSON：snapshot + strategy
```

### 3.3 L1/L2 自适应 PSO（E3c）

在 `src/models.py` 或新建 `src/pso_profile.py`：

```python
def select_pso_profile(snapshot: MarketSnapshot) -> dict:
    spread_pct = (snapshot.l1.eth_usdc_price - snapshot.l2.eth_usdc_price) / snapshot.l2.eth_usdc_price
    if spread_pct > 0.01:
        return {"num_particles": 1500, "max_iter": 120}  # 大价差：广搜
    if snapshot.l2_lag_ms < 500:
        return {"num_particles": 500, "max_iter": 60}    # 低延迟：快搜
    return {"num_particles": 1000, "max_iter": 100}      # 默认
```

`main.py` 中 `run_pso_on_snapshot` 调用此函数覆盖 `PSO_CONFIG`。

## 6. 常见问题

| 问题 | 处理 |
|------|------|
| 本地 CUDA 不可用（RTX 50 系） | 本地用 CPU；E2 只在云主机跑 |
| ETH WebSocket 超时 | `main.py --live --http` |
| `torchrun` 找不到 | `pip install torch` 且需在 Linux 多卡机 |
| NCCL 报错 | 确认多卡在同一节点；`export NCCL_DEBUG=INFO` 排查 |
| Streamlit 端口占用 | `streamlit run viz/dashboard.py --server.port 8502` |
| Mock 无套利 | 用 `--record-id snap_005` 等已知有价差记录 |
